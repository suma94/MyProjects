Button names are classes.USE these class names to create buttons




TextBox and TextArea classes are in text area




NOTE:To align the textboxes ,create each text box in individual <div> tag and give class="inputwrap".Sample Code is below

<div class="inputwrap">
<label class="vz-standard-font">Answer</label>
<input type="text" class="vztext" name="answer">
</div>
<div class="inputwrap">
<label class="vz-standard-font">Password</label>
<input type="password" class="vztext" name="password" required >
</div>
